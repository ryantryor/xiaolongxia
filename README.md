## 🌟 什么是小龙虾？

**OpenClaw 小龙虾** 是一个专注于 OpenClaw 生态的中文聚合站与学习平台。我们致力于帮助国内开发者、AI 爱好者以及零基础小白，**从零开始，7天掌握最具潜力的全球热门开源 AI 平台**。

在这里，AI 不仅能陪你“聊天”，更能真正帮你“做事”——连接本地物理机、响应即时通讯指令、全自动处理工作流。

## 🛠️ 本地开发

想在本地运行或参与构建 `xiaolongxia.vip`？非常简单：

```bash
# 1. 克隆代码仓库
git clone https://github.com/ryantryor/xiaolongxia.git

# 2. 进入目录
cd xiaolongxia

# 3. 启动本地服务器 (需安装 Node.js环境)
npx serve .
```

打开浏览器访问 `http://localhost:3000` 即可预览！
